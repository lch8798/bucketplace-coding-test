# bucketplace-coding-test

버킷플레이스 과제 프로젝트입니다.

## [Live](http://leechan.kr)

# 시연

### Install

```
yarn install
```

### Run

```
yarn start
```

# 기술 스택

### Setup

-   CRA (create-react-app)
-   typescript
-   craco (alias)

### React

-   react
-   redux
-   react-redux
-   redux-thunk
-   styled-components

### Utils

-   axios (HTTP client lib)
